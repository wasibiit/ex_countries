      ExCountries.Country (ex_countries v0.0.3)

      This module contains functions to transform country codes into country names and vice versa

      Summary

      Functions

      code_by_name(country_name, code \\ "alpha2")

      Returns country code(alpha2 or alpha3) by full name and returns nil if not found.

      normalize_alpha(country_name_or_code, opts \\ :switch)

      Normalize country code(alpha2 or alpha3) and returns the other codes. By default it will switch between alpha2 and alpha3, but you can pass :alpha2 or :alpha3 options, will only return alpha2 or alpha3.
